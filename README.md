# Zyclos

Zyclos é uma plataforma digital de moda sustentável para anunciar, comprar, vender e trocar roupas usadas. O projeto foi criado como MVP para Trabalho de Conclusão de Curso do ensino médio técnico em Informática.

## Tecnologias

- ReactJS
- Vite
- React Router DOM
- JavaScript
- CSS puro
- LocalStorage

## Como executar

```bash
npm install
npm run dev
```

## Funcionalidades do MVP

- Página inicial responsiva
- Login e cadastro simulados
- Catálogo com pesquisa e filtros
- Detalhes da roupa
- Criação de anúncio com imagem simulada
- Favoritos
- Perfil do usuário
- Painel de impacto sustentável
- Chat interno simulado

## Melhorias futuras
